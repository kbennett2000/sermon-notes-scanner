MakeACopy 3.5.0 (Release-Candidate)

* New: Inbox Mode – configure a default export folder for frictionless scanning, no file picker needed (ideal for paperless-ngx, Nextcloud, Syncthing workflows)
* Fixed: File picker now remembers the last used folder correctly (improved initial URI handling for better OEM compatibility)
* Fixed: Crash (OutOfMemoryError) during JPEG-compressed PDF export caused by alpha channel extraction in large images
* Fully offline, no cloud or tracking

Hint: This is a release candidate - feedback and bug reports are welcome!
